## About The Project

A command-line application that manages a company's employee database. It uses Node.js, Inquirer, and MySQL.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/topcowmoo/fluffy-disco-note-taker
   ```
2. Install NPM packages
   ````sh
   npm install   ```
   ````
